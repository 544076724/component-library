```javascript
<template>
    <div style="width:100px;height:100px">
        <c-dhn-lottie
            :options="option"
        >
        </c-dhn-lottie>
    </div>
</template>
<script>
import jsonData from './data.json'
export default {
  data () {
    return {
        return {
          option: {
            loop: true,
            autoplay: true,
            animationData: {
                ...jsonData,
                assets:jsonData.assets.map(item => {
                  const { u, p, ...args } = item
                  return p ? { ...args, p: require(`./images/${p}`)} : item
                })
            }
          }
        }
    }
  },

}
</script>
```

上面 json 中的 u,p 是路径相关的，我们这里为图片加载做了下路径处理, 如果 json给提供了图片，可以根据自己需求更改引用路径

该组件使用了 lottie-web 库,详细用法也可以参考它
