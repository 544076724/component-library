# DateCountdown 倒计时

#### 基本使用说明  左右展示

<Preview>
<Story name='basic'>{{ 
  components: { CDhnDateCountdown }, 
  template: `<c-dhn-date-countdown 
            :dayLang="'日'"
            :countdownTime="4000"
            >
            </c-dhn-date-countdown>`, 
}}</Story> 
</Preview>

```javascript
<template>
    <c-dhn-date-countdown  dayLang="日" :countdownTime="time"></c-dhn-date-countdown>
</template>
<script>
  export default {
    data () {
      return {
        time:4000 //注意 这里是以秒 为单位   
      }
    }
  }
</script>
```
